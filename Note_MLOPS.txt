Tài lieu hoa

$ poetry run pdoc --docformat=google --output-directory=docs/ src/bikes

Trong lệnh $ poetry run pdoc --docformat=google --output-directory=docs/ src/bikes, input và output thay thế ở đây là:
Input: src/bikes định ra thư mục hoặc module nguồn mà pdoc sẽ đọc để tạo tài liệu. Đây là phần đầu vào của lệnh where pdoc sẽ phân tích và tạo tài liệu từ mã nguồn có trong src/bikes.
Output: --output-directory=docs/ chỉ định thư mục đích nơi pdoc sẽ lưu trữ các tài liệu được sinh ra. Kết quả hay output của lệnh này sẽ được lưu vào thư mục docs/.

#install pre-commit
#cài đặt các Pre-Commit Hooks để sử dụng công cụ pre-commit để thiết lập các hooks

poetry add -G commits pre-commit

poetry run pre-commit install --hook-type pre-push
poetry run pre-commit install --hook-type commit-msg

#Thực thi Pre-Commit Hooks: Bạn có thể tự động chạy các hooks trước mỗi lần commit hoặc push, hoặc kích hoạt chúng thủ công với:

poetry run pre-commit run

--> lệnh trên giúp kiểm ra các điều kiện trức khi push về các file có trong file .pre-commit-config.yaml
